오늘 공부하면서 느낀 점

  • 3.3.1 자바스크립트의 동적 스코프 부분에서

    다행히 this 레퍼런스를 call 이나 apply 로 넘겨주지 않았거나 null로 바인딩했다면 동적 스코프와 혼동이 발생하지 않는다.

    라고 나와 있는데 null을 바인딩한다면… 코드가 머리에 그려지지 않는다. 내일 이 부분을 한 번 더 읽어 봐야 겠다.
    그리고 call 과 apply 의 실행컨텍스트 this 바인딩과 스코프 체이닝에 대해서도 좀 더 자세히 찾아 봐야겠다.

  • 3.3 동적 스코프의 예제에서 makeBindFunc 함수 예제를 봤을 때 이해가 되지 않았는데 차분히 읽으면서 머리 속으로 컴파일 해 나가는 재미가 있었다.
    내일은 직접 타이핑 하면서 다양하게 시도해봐야겠다.
  • 오전, 오후를 동계 올림픽 보느라 오늘은 계획 했던 만큼 공부를 하지 못했다. 내일은 설날이므로 이동 시간을 이용해서 시간분배를 잘 해야겠다.
