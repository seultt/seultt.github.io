React-JSX

React JSX

React.js는 일반 JavaScript문법이 아닌 JSX 문법을 사용하여 UI 템플릿화 한다.
JSX를 사용하는 것은 필수가 아니지만 이를 사용하면 다음과 같은 장점이 있다.

  • JSX는 컴파일링 되면서 최적화 되므로 빠르다.
  • Type-safe 이며 컴파일링 과정에서 에러를 감지 할 수 있다.
  • HTML에 익숙하다면, JSX를 사용하여 더 쉽고 빠르게 템플릿을 작성 할 수 있다.

If-Else 문 사용 불가

JSX 안에서 사용되는 JavaScript 표현에는ㄴ IF-Else 문이 사용이 안된다.
